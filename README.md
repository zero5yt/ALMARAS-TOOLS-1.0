# ALMARAS TOOLS v1.0

Termux video toolkit.

Features:
- Download Facebook Video
- Combine Video + Audio
- Convert MKV to MP4
- Cut Video
- Add Subtitle

Install:

pkg update -y
pkg install git -y
git clone https://github.com/zero5yt/ALMARAS-TOOLS-1.0/tree/main
cd almaras-tools
bash install.sh
bash almarastools.sh
