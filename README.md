# 🚀 ALMARAS-TOOLS V1.0

<p align="center">
  <i>The Ultimate Telegram Upload & Video Tool for Termux</i>
</p>


### 🛠 INSTALLATION

bash
pkg update && pkg upgrade -y && pkg install git -y && git clone https://github.com/zero5yt/ALMARAS-TOOLS-1.0 && cd ALMARAS-TOOLS-1.0 && bash install.sh
🔑 HOW TO SETUP

Go to my.telegram.org and login with your phone number.

Go to "API development tools" and create a new App to get your API ID and API HASH.

Run the tool:

code
Bash
download
content_copy
expand_less
./almarastools.sh

Enter your API ID, API HASH, Phone Number, and Verification Code when prompted.

💡 FEATURES

🎥 Download: Download Facebook Videos directly.

✂️ Video Tools: Combine M4S/MKV/Audio, Cut Videos, Add Subtitles.

🚀 Fast Upload: Bypass 50MB Bot API limit using Telethon (MTProto).

⚡ Dual Upload Mode:

Video Mode: Upload with thumbnail/preview.

File Mode: Direct file upload (faster, no compression).

🛡 Security: Smart Login system (Your session data stays on your device).

⚠️ IMPORTANT NOTES

Storage Permission: Make sure to run termux-setup-storage in your Termux before using this tool.

Security: This tool uses your personal Telegram account to bypass limits. Do not share your user_session.session or config_api.txt files with anyone.

Admin: If using for a channel, ensure your account has permission to post messages.
